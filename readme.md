# Geospatial Atlas Apps

This is the Geospatial Atlas Apps repository showcasing various apps built with the tool. Explore huge geospatial datasets interactively in your browser!

## Apps

### GlobalGeoTree

GlobalGeoTree is a comprehensive global dataset for tree species classification, comprising 6.3 million geolocated tree occurrences spanning 275 families, 2,734 genera, and 21,001 species across hierarchical taxonomic levels. Each sample is paired with Sentinel-2 image time series and 27 auxiliary environmental variables.

Authors: Yang Mu, Zhitong Xiong, Yi Wang, Muhammad Shahzad, Franz Essl, Holger Kreft, Mark van Kleunen, Xiao Xiang Zhu

Publication: https://arxiv.org/abs/2505.12513

Datasource: https://huggingface.co/datasets/yann111/GlobalGeoTree
